# Licensing — read before any public release

**Status: UNRESOLVED. Do not add an open-source LICENSE file or publish publicly until
this is settled.**

FACET was built in a work context. Tools created as part of employment are commonly the
**employer's** intellectual property, not the author's, regardless of who typed the code
or where it is hosted. This is not legal advice — confirm with the employer (and, if
needed, a lawyer) **before** any public release or open-sourcing.

Open questions to resolve first:
1. Who owns this work-product? (Likely the employer.)
2. If cleared, what license? (MIT is a common, permissive choice — but only the owner can
   grant it.)
3. Have all real configs, hostnames, IPs, site names, and any depiction of the
   employer's actual security architecture been removed? (Built-in samples are already
   anonymized; verify nothing real has crept in.)

Until 1–3 are answered, treat this repo as **internal / unlicensed** and do not
distribute it outside the team.
